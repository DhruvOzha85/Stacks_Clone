# 📚 Stacks Clone

A static front-end clone of the **Stacks** developer community platform, built with pure HTML and CSS. This project recreates the look and feel of the Stacks website — a platform for developers to explore, build, and learn together.

🔗 **Live Demo:** [clonestacks.netlify.app](https://clonestacks.netlify.app/)

---

## 📁 Project Structure

```
Stacks_Clone/
├── index.html       # Landing / Home page
├── explore.html     # Explore page
├── build.html       # Build page
├── learn.html       # Learn page
└── styles.css       # Global stylesheet
```

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure & content (65.3%) |
| CSS3 | Styling & layout (34.7%) |

---

## ✨ Features

- Multi-page static website with 4 pages (Home, Explore, Build, Learn)
- Responsive layout styled to match the Stacks UI
- Clean navigation structure across all pages
- Deployed and publicly accessible via Netlify

---

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/DhruvOzha85/Stacks_Clone.git
   ```
2. Open `index.html` in your browser — no build step or dependencies required.
